# Project Brief

## Overview
Beauty Recycle is a web application designed to incentivize the recycling of beauty product packaging. It uses gamification (points, rewards) to encourage users to scan and recycle materials like plastic, glass, and cardboard.

## Core Goals
- Allow users to scan beauty product packaging.
- Track environmental impact (CO2 saved, plastic recycled).
- Reward users with points redeemable for rewards.
- Provide a dashboard for tracking activity.

## Tech Stack
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **API:** tRPC
- **Database:** Prisma (PostgreSQL likely, referenced in file structure)
- **Auth:** Clerk (implied by `(auth)` folder and clerk package)
