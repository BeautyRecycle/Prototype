# Tech Context

## Technologies
- **Frontend:** Next.js, React, Tailwind CSS, Lucide React (icons).
- **Backend:** Next.js API Routes (via tRPC).
- **Database:** Prisma ORM.
- **Linting:** ESLint with strict rules (some fixes applied).

## Development Setup
- **Package Manager:** pnpm.
- **Build Command:** `pnpm run build`.
- **Dev Command:** `pnpm dev`.

## Constraints
- Strict TypeScript and ESLint rules are enforced.
